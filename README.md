# 🌍 YatraMitra – Smart & Safe Travel Companion Finder

## 🚀 About the Project

YatraMitra is a web-based platform built to help people find **the right travel companions**, not just random ones.

Many people love traveling but hesitate because they don’t have the right company or feel unsafe connecting with strangers online. This project aims to solve that problem by making travel **smarter, safer, and more social**.

---

## 🎯 Why We Built This

While exploring existing travel apps, we noticed some common issues:

* People couldn’t trust other users
* Matching was very basic (only location-based)
* Safety features were almost missing
* Many profiles were inactive or fake

So, we decided to build a platform that focuses on **trust, compatibility, and safety first**.

---

## 💡 What Makes YatraMitra Different

Instead of just showing random travelers, YatraMitra helps you find **people who actually match your vibe**.

It uses:

* 🤖 Smart matching based on interests, budget, and personality
* 🛡️ Trust score to identify genuine users
* 🚨 Safety features to make users feel secure

---

## ✨ Key Features

### 🔐 Login & Authentication

Users can sign up using email or Google and securely access their accounts.

---

### 👤 User Profile

Each user can create a detailed profile including:

* Interests and travel preferences
* Budget range
* Travel style (luxury/backpacker)
* Personality type
* Profile photo and intro

---

### 🤖 Smart Matching (Main Feature)

The system calculates a **compatibility score** between users based on:

* Interests
* Budget
* Travel dates
* Personality
* Reviews

This helps users connect with people who are actually suitable travel partners.

---

### 📍 Trip Planning

Users can:

* Create trips
* Explore trips created by others
* Find people traveling to the same destination

---

### 💬 Chat System

Users can chat in real-time once matched, making communication simple and quick.

---

### 🛡️ Trust & Verification

To reduce fake profiles:

* Users can verify themselves
* A **Trust Score** is assigned based on activity and reviews

---

### 🚨 Safety Features

User safety is a priority:

* SOS button
* Emergency contacts
* Safety mode

---

### 💸 Expense Splitting

Users can manage trip expenses and split costs easily within the app.
